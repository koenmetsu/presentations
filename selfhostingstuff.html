Self-hosting

Oh wait!

Like an application that hosts its web server in its own process?

Oh right!

Like how you use a program as part of the toolchain that produces new versions of that same program?

Selfhosting is the process of

locally hosting and managing applications

instead of renting from SaaS providers.

What can you self-host?

Feed readers

Bookmarking services

ReadItLater clones

Git servers

Automation software

Image galleries

Wikis

Mindmapping software

Social networks

Media streaming

File sharing

DNS services

Monitoring software

Pastebins

CI tools

Password managers

Maps

Search engines

Gee... that sounds like a lot of work...

Advantages

Decentralize the web

Learn stuff

Own your own data => Privacy++

Only you can shut yourself down

Fun

Disadvantages

Security is important

Maintenance

You can shut yourself down

Time sink

Tips

Docker

Internal hosting: NUC

External hosting: DigitalOcean

SSL: Letsencrypt

(Wildcard) Dyndns: Synology

Reverse proxy: nginx

Security: Basic auth through nginx

Start with non-crucial stuff
